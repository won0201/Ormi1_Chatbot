- 주제 : 고민 상담 챗봇
- 설명 : 기존 ChatGpt의 기능을 살려, 고민을 이야기하면 해결책을 알려주거나 공감해주는 역할의 챗봇이다.
- 구현방식
  1) 흔히 카카오톡 플러스 친구에 나오는 챗봇과 유사한 형태를 지니도록 한다.
  	  * 서로 대화할 수 있는 방식

- 수정사항
* textarea 모양 수정 O
* submit 위치 수정 O
* question 우측 여백 없애기 O
* 말풍선 화살표 붙이기 O
* question, answer 위로 옮기기 O
* answer 부분 위치 수정
* question 말풍선 크기 바뀌는거 고치기
* answer 부분 위치 수정
* question 부분도 chatbot 형태로 만들기 
* 말풍선 크기 조절
* 전체적인 꾸미기
* 로딩 화면 만들기
* 추가 기능 생각하기
